# Sensitive Grand Piano · Professional Edition

Un instrumento virtual de piano diseñado con precisión para la línea **Sensitive**. Enfocado en la fidelidad sonora y la respuesta táctil inmediata, ideal para producción musical en entornos web.

## 🎹 Especificaciones Técnicas
- **Core Engine:** Integración nativa con `Tone.js` (Web Audio API).
- **Latency:** Optimizado para una respuesta de latencia mínima (Near-Zero Latency).
- **MIDI Input:** Detección automática de controladores MIDI externos.
- **FX Chain:** Cadena de efectos integrada para modelado de estudio analógico.
- **Visualizador:** VU Meter en tiempo real para monitoreo de ganancia.

## 🚀 Despliegue (Deploy)
Este proyecto es una aplicación web estática (Single File App). Se puede desplegar instantáneamente en:
- **Vercel / Netlify:** Solo necesitas conectar el repositorio y el sistema detectará el `index.html`.
- **GitHub Pages:** Habilita la opción "Pages" en la configuración de tu repositorio y apunta a la rama `main`.

## 🛠 Modo de uso
1. Abre la URL del proyecto en un navegador moderno (Chrome/Edge recomendado).
2. Haz clic en **"Iniciando motor"** para activar la síntesis sonora.
3. Conecta tu controlador MIDI o utiliza el teclado de la pantalla.

## 👤 Autor
Desarrollado por **Juan Guillermo Coronel** para el ecosistema de hardware musical **Sensitive**.
